# WinRay 配布

**WinRay** の配布物（単一実行ファイル）と自動更新の配信元リポジトリです。
このリポジトリには**ソースコードは含まれません**（成果物のみ）。

WinRay は Windows 常駐型の「グラス調ショートカットランキー」ランチャーです。半透明のすりガラス
（Acrylic）デザインで、登録したアプリ・ファイル・フォルダ・URL・コマンドをカテゴリ分けして
すばやく起動します。

> このアプリは**広告付き**プロダクトです。パネル内に控えめな広告が表示されます。

## ダウンロード / インストール

1. [Releases](https://github.com/lighfu/WinRay-dist/releases/latest) から最新の `WinRay.exe` を入手します。
2. 実行するだけで起動します（インストーラ不要・単一ファイル・自己完結）。
3. システムトレイに常駐します。トレイメニューから設定・終了ができます。

## 自動更新

WinRay は起動時にこのリポジトリの最新リリースを確認し、新しいバージョンがあればトレイで通知します。
承認すると新しい実行ファイルをダウンロードし、自動で入れ替えて再起動します。
手動確認はトレイメニューまたは設定から行えます。

## 動作要件

- Windows 10 / 11（主ターゲットは Windows 11。Acrylic/Mica はフォールバック対応）
- x64
- .NET ランタイム不要（自己完結発行）

## リリース

配布物はメンテナが `build-deploy` で生成した難読化済み単一 exe を、タグ `vX.Y.Z` の
GitHub Release に添付します。

## ライセンス / プライバシー

WinRay 本体は商用プロプライエタリ製品です。本リポジトリは配布と更新のためのものです。
広告表示に伴い、匿名の表示/クリックイベントが Google Analytics に送信されることがあります
（個人を特定する情報は送信しません）。
